Главный модуль

rb!help - Отобразить список доступных команд или информацию о конкретной команде

rb!idea - Предложить идею для развития сервера

rb!key - Сгенерировать ключ доступа к панели управления

rb!ping - Отобразить текущую задержку

rb!setcolor - Получить цвет в формате HEX (#ffffff)

rb!avatar - Отобразить аватарку

Информационный модуль

rb!info - Отобразить информацию о боте

rb!link - Получить не истекающее приглашение

rb!staff - Отобразить список персонала сервера

Модуль для общения

rb!profile - Отобразить профиль пользователя

rb!top [rep, cash] - Отобразить топ пользователей по уровням [репутации или балансу]

Экономический модуль

rb!buy - Купить указанную роль

rb!cash - Отобразить ваш текущий баланс

rb!daily - Получить ежедневный бонус

rb!pay - Перевести валюту другому пользователю

rb!shop - Отобразить список доступных для покупки ролей

rb!work - Подзаработать немного денег

Другое

rb!guild - Отобразить информацию о сервере

rb!role - Отобразить информацию о роли

rb!user - Отобразить информацию о пользователе
